# 🎂 Calculadora de Idade

Um projeto prático que calcula a idade exata do usuário com base na data de nascimento. Desenvolvido para exercitar lógica de programação e manipulação do DOM.

## 🚀 Funcionalidades

- **Cálculo Preciso**: Resolve o problema de "ano a mais" antes do aniversário.
- **Design Responsivo**: Adaptado para celulares e desktops.
- **Efeitos Visuais**: Botões com feedback de clique e hover.
- **Auto-limpeza**: Limpa os campos automaticamente após o cálculo.

## 🛠️ Tecnologias

- **HTML5**
- **CSS3** (Flexbox e Media Queries)
- **JavaScript** (Objeto Date, Math.floor e EventListeners)

## 📖 Como usar

1. Insira sua data de nascimento.
2. Clique no botão de calcular.
3. O resultado aparecerá na tela e sumirá após alguns segundos.
